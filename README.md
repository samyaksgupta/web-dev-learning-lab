# Web Development Learning Lab 🚀

An interactive, beginner-friendly web application designed to teach the fundamentals of web development as per the academic syllabus.

## 🌟 Key Features
- **Interactive OOP Playground**: A banking system simulation demonstrating Encapsulation, Inheritance, and Classes.
- **Client-Server Visualization**: Real-time animation of HTTP packets traveling between client and server.
- **Live Protocol Terminal**: Both in-browser and system terminal logging for HTTP, DNS, TCP, and DOM events.
- **Code Tooltips**: Hover over UI components to see the HTML/CSS code behind them.
- **Syllabus Coverage**: Includes Bootstrap, WWW Protocols, JS Basics, ES6, AJAX, XML, and JSON.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6), Bootstrap 5.
- **Backend**: Node.js, Express (for real-time system terminal logging).

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repo-url>
   cd web-dev-learning-lab
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the server**:
   ```bash
   node server.js
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:3001` (or the port specified in the terminal).

## 📚 Syllabus Concepts Covered
- **Bootstrap**: Grid, Jumbotron, List Groups, Pagination, Icons.
- **WWW**: DNS resolution, Hosting types, Client-Server model.
- **JavaScript**: DOM/BOM manipulation, Form Validation, ES6, OOP.
- **Data**: AJAX fetch, JSON parsing, XML/XSLT transformation.
